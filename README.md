# Medical Image Processing using OpenCV

This project demonstrates a basic medical image processing pipeline using Python and OpenCV on chest X-ray images.

## 🔧 Techniques Used
- Grayscale Conversion
- Gaussian Blur (Noise Reduction)
- Edge Detection (Canny)
- Image Segmentation (Thresholding)

## 📸 Output
The project processes an X-ray image and displays:
- Original Image
- Grayscale Image
- Blurred Image
- Edge Detection Output
- Segmented Image

## 🛠 Tools & Libraries
- Python
- OpenCV
- NumPy
- Matplotlib
